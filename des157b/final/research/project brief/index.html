Projectbrief

Project summary:

This project will focus on providing free textbook resources for students from different social classes to connect with the same education.

Project Goals:

The goal of this project is to create a community where users can obtain free text books in PDF and also users can swap textbooks directly with other users of the platform without having to pay. So that everyone can access the same opportunity in education by providing free resources for textbooks. Additionally I want to create an opportunity for users to meet new people when they exchange textbooks with other users.

Design Strategies:

The overall layout of the page will be friendly, simple and with a warm and bright color scheme. In terms of typography, I plan to use a Serif font inspired by the classical vibe that many old books have and paired it with a sans serif font for easy reading and also to enhance the modern look. Finally, there will be a landing page like a tutorial and let the user know what they can achieve with the website. The main page will display a 3D gallery with a different textbook cover where users can look around the textbook and click the category menu to access different disciplines they are interested in looking at.

Interaction Strategies:

For the Interaction part, the free PDF link will display after the user clicks on the textbook. And also users can see who wants to swap this textbook in the same page. Once the user clicks the swap button, he/she will upload textbooks photos which he/she wants to swap and ask for exchange. So they can start swapping the textbooks at a ratio 1:1 and also users can add the textbooks to this swap page also where other users can see this post.
