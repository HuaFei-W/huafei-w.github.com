Arrow functions with object literal

Named Function

            function sayName() {
                return {
                    name: "Amy Yu",
                    age: 20,
                }
            }
            console.log(sayName().name);

Function Expression

sayName() is a function expresssion that returns an object that returns an object that has properties name and age set to “Amy Yu” and “20”, respectively. 

            const sayName = function(){
                return {
                    name: "Amy Yu",
                    age: 20,
                };
            }
            console.log(sayName().name);

Arrow Function

            const sayName = () => ({
                name: "Amy Yu",
                age: 20,
              });
            console.log(sayName().name);
